## 📊 أنواع الأخطاء المغطاة

### 1️⃣ أخطاء قاعدة البيانات (Database Errors)
- ✅ حسابات رئيسية غير موجودة
- ✅ حسابات فرعية غير موجودة
- ✅ بيانات مرجعية مفقودة
- ✅ تكرار القيم الفريدة
- ✅ حقول مطلوبة فارغة

### 2️⃣ أخطاء الفترات المحاسبية (Accounting Period Errors)
- ✅ فترة محاسبية مغلقة
- ✅ عدم وجود فترة مفتوحة
- ✅ تاريخ خارج نطاق الفترة

### 3️⃣ أخطاء التحقق من البيانات (Validation Errors)
- ✅ كميات غير صحيحة (≤ 0)
- ✅ أسعار غير صحيحة (≤ 0)
- ✅ عناصر فارغة في الفواتير
- ✅ عملاء/موردين غير محددين للمبيعات/المشتريات الآجلة

### 4️⃣ أخطاء الترحيل (Posting Errors)
- ✅ فشل إنشاء قيد يومية
- ✅ فشل ربط الفاتورة بالمخزن
- ✅ فشل تحديث حركات الحسابات

```dart
// - كود خطأ قابل للتتبع
// - معالجة مركزية ممكنة
// - تجربة مستخدم محسنة
```

## 🔄 التكامل مع GlobalErrorHandler

جميع الأخطاء الآن يتم التقاطها ومعالجتها بواسطة `GlobalErrorHandler`:

```dart
// في الـ UI
GlobalErrorHandler().catchFailureAsync(() async {
  await accountingService.postSale(sale, items);
}).then((either) {
  either.fold(
    (failure) => showErrorSnackBar(failure.message), // رسالة عربية واضحة
    (success) => showSuccessSnackBar('تم الترحيل بنجاح'),
  );
});
```
